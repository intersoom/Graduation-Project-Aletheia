# Aletheia: Little Prince
### 작품설명
'Aletheia'는 그리스어로 탈은폐, 즉 진리를 뜻합니다. 사람들에게 친숙한 어린왕자 스토리에 하이데거 철학을 녹여낸 3D 웹 미디어아트로 작품을 통해 대지와 세계의 투쟁 그리고 진리의 찰나를 경험합니다.
